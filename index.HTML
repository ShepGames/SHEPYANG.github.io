Introduction


SHEP AI

Advancing the evolution

The AI matrix is a system which collects an input, checks its data and outputs the found string of words to the user. If the input is not found in the data, the AI will ask the user to add it. The input will then be in the solid state memory and accessible in the future when the AI is faced with this input. The output can be from saying words to physically learning through potentiometers and servos. The AI has artificial moods, for example; if you use negative vocabulary the AI’s mood variable will be equal to sad, this means it will not have access to the main memory but have access to the sad part of its ‘brain’.
